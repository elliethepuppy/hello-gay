# hello-gay

a little greeter that will say whatever you want! or, yknow..

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     hello-gay:
       github: elliethepuppy/hello-gay
   ```

2. Run `shards install`

## Usage

```crystal
require "hello-gay"
```

use it however you see fit
